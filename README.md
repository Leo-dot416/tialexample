# tialexample

Student Coding Projects Repository

## Overview
This repository contains 3 coding projects designed for students to practice programming skills.

## Projects

### Project 1: Hello World Application
**Location:** `project1/`  
**Difficulty:** Beginner  
**Concepts:** Basic functions, unit testing, error handling  

A simple introduction to programming with a Hello World application.

### Project 2: Calculator Application
**Location:** `project2/`  
**Difficulty:** Intermediate  
**Concepts:** Multiple functions, error handling, user interfaces, comprehensive testing  

Build a command-line calculator that performs basic arithmetic operations.

### Project 3: To-Do List Application
**Location:** `project3/`  
**Difficulty:** Advanced  
**Concepts:** Object-oriented programming, file I/O, data persistence, multiple modules  

Create a full-featured to-do list application with data persistence.

## Getting Started

1. Choose a project based on your skill level
2. Navigate to the project directory (e.g., `cd project1`)
3. Read the project's README.md file for detailed instructions
4. Implement the required functionality
5. Run the tests to verify your implementation

## Structure

Each project includes:
- `README.md` - Project description and requirements
- Source code files with TODO comments
- Test files for verification
- Clear learning objectives

Happy coding!
