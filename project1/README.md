# Project 1: Hello World Application

## Description
Create a simple Hello World application that demonstrates basic programming concepts.

## Requirements
- Create a function that prints "Hello, World!" to the console
- Add error handling
- Include unit tests

## Files
- `main.py` - Main application file
- `test_main.py` - Unit tests for the application

## Getting Started
1. Implement the `hello_world()` function in `main.py`
2. Run the tests with `python test_main.py`
3. Make sure all tests pass

## Learning Objectives
- Understanding basic function creation
- Learning about unit testing
- Error handling basics
